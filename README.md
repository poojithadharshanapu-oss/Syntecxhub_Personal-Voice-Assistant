# 🎙️ Personal Voice Assistant

## 📌 Project Overview

The Personal Voice Assistant is an AI-based Python project that allows users to interact with a computer using voice commands. The assistant listens to user instructions, converts speech into text, processes commands using rule-based logic, and responds using text-to-speech technology.

This project demonstrates the practical implementation of speech recognition, automation, and AI-based human-computer interaction.

---

## 🚀 Features

- 🎤 Voice-based user interaction
- 🗣️ Speech-to-text conversion
- 🔊 Text-to-speech response
- 🌐 Open websites using voice commands
- ⏰ Tell current time
- 📝 Open Notepad
- 🧮 Open Calculator
- ❌ Exit assistant using voice command
- ⚡ Error handling for unrecognized commands

---

## 🛠️ Technologies Used

- Python
- SpeechRecognition
- Pyttsx3
- SoundDevice
- SciPy
- Webbrowser Module
- OS Module
